# AI陪伴型寵物｜最懂你生活節奏的AI室友


## 一、設計動機

在高壓與孤獨成為常態的現代生活中，許多年輕人面臨心理壓力、社交疏離與自我管理困難。傳統AI設備缺乏情感溫度，無法真正陪伴使用者；真實寵物雖然可帶來溫暖與陪伴，卻因公寓規定與金錢壓力難以實現。希望能設計一款AI寵物，根據使用者每天的行為舉止，分析生活模式，提供個人化的情緒支持與陪伴，減輕孤獨，增添生活動力。

## 二、設計核心理念

- 打造一款能主動感知、回應使用者情緒與生活狀態的AI寵物裝置  
- AI 不再只是接收指令，而能主動觀察並提出關心與建議  
- 陪伴模式以日常作息為基礎進行互動   
- 目標是減輕壓力、穩定情緒、提升生活幸福感與動力


## 三、應用情境

- 獨居年輕職場新人，工作壓力大、社交有限，需要情緒支持  
- 早晨醒來時，AI寵物主動互動，提供溫暖陪伴  
- 工作期間，透過行為監測提醒休息，避免過度疲勞  
- 晚間情緒低落時，AI主動聊天安慰，提供陪伴與鼓勵   


## 四、核心功能

| 功能模組         | 說明                                         |
|------------------|----------------------------------------------|
| 行為感知         | 透過感測器與Arduino裝置監測使用者日常行為       |
| AI情緒分析       | 利用AI分析使用者情緒與生活模式，提供即時反饋     |
| 個人化情緒支持   | 根據分析結果提供溫暖的語音回饋與互動              |
| 主動提醒         | 主動推播休息、放鬆等生活建議，減輕壓力與孤獨感   |
| 硬體互動         | Arduino控制燈光、聲音等裝置，增加陪伴感與互動性   |

## 五、互動場景設計

| 時間區段           | 情境模擬                                   | 互動描述                                                                                                                               |
|--------------------|--------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------|
| **早上（6:00-10:00）**       | 莫生人平日都差不多7:30起床<br>AI分析使用者「過去每日起床時間（7:30±15分鐘）」 | **正常情況**:<br>AI感知「偵測到使用者起床的動作，心跳與呼吸頻率，得知使用者已甦醒」<br>AI寵物會「微震動，尾巴（伺服馬達）快搖模擬寵物興奮的情緒，眼睛(LED)亮綠燈，與使用者一同起床(開機)」<br><br>**沒起床狀態**:<br>AI於每日平均起床時間未感知到「使用者起床的動作，心跳與呼吸頻率，得知使用者未甦醒」<br>AI寵物會「微震動，眼睛(LED)亮燈，並主動出聲提醒使用者該起床上班」 |

| **下午（14:00-18:00）**| 莫生人在家遠端工作<br>AI分析使用者「工作習慣或靜坐過久，提醒放鬆」 | **久坐提醒**:<br>AI感知「偵測到使用者久坐未起來活動的行為，得知使用者已靜坐太久」<br>AI寵物會「提醒使用者該休息活動」<br><br>**提醒放鬆**:<br>AI感知「偵測到使用者在同個位置使用電腦時間已超過45 ~ 50分鐘，得知使用者已用眼過度」<br>AI寵物會「慢慢增強震動，尾巴（伺服馬達）搖晃甩模擬寵物不悅的情緒，眼睛(LED)亮紅燈出聲提醒使用者該休息10~15分鐘，並於休息時間結束後停止震動與尾巴動作，眼睛(LED)亮綠燈提醒使用者該繼續工作，避免怠惰」 |
| **晚上（18:00-22:00）**| 莫生人孤單寂寞，工作壓力無人傾訴<br>AI分析使用者「情緒/話語並給予情緒價值」 | **情緒價值**:<br>AI感知「偵測到使用者情緒低落，得知使用者情緒不好」<br>AI寵物會「尾巴（伺服馬達）緩慢搖晃甩模擬寵物不安的情緒，眼睛(LED)亮黃燈主動找使用者聊天，分析使用者遇到的問題並提出建議」<br>使用者可以透過AI寵物得到情緒價值與建議 |
| **半夜（22:00-6:00）** | 莫生人上床睡覺<br>AI分析「睡眠品質、放鬆與引導入睡」         | **正常情況**:<br>AI感知「使用者靠近床，並偵測到使用者翻身頻率降低，心跳與呼吸頻率達睡著後的狀態」<br>AI寵物「停止動作，閉上眼睛(LED燈)模擬寵物入睡(休眠)」<br><br>**失眠**:<br>AI感知「偵測翻身頻率上升，心跳與呼吸頻率未達睡著後的狀態」<br>AI寵物會「釋放出助眠香氛(如薰衣草)或播放助眠輕音樂」幫助使用者放鬆入眠。 |



## 六、實作結構（架構設計）

**感知模組**  
   - 使用心跳、呼吸頻率感測器偵測使用者生理狀態  
   - 使用動作感測器與姿態偵測分析行為模式  

**數據收集與分析**  
   - 收集使用者日常行為數據（起床時間、用眼時間、情緒狀態等）  
   - 利用AI模型分析生活習慣與情緒趨勢，生成個人化回饋  

**互動控制**  
   - Arduino控制伺服馬達（模擬寵物尾巴搖擺）  
   - LED燈光變化表示情緒狀態（綠燈、黃燈、紅燈）  
   - 振動馬達提供觸覺反饋  

**語音輸出**  
   - 使用語音合成（TTS）系統播放AI寵物回應與提醒  
   - 可選擇搭配語音辨識系統實現簡單對話  

**系統整合與資料管理**  
   - 由Python主程式統籌數據收集、AI分析與控制訊號輸出  
   - 所有資料本地儲存，保障隱私安全

## 十、專案特色

- **情感感知互動**：結合多種感測器，真實感知使用者生理與情緒狀態。  
- **個人化陪伴**：AI根據使用者生活習慣與情緒狀態，提供專屬支持與提醒。  
- **多模態反饋**：透過振動、燈光與語音多種方式提供情緒互動體驗。  
- **離線運作**：所有資料本地處理與儲存，保障使用者隱私與資料安全。   
- **適用現代生活**：專為獨居職場新人設計，緩解孤獨與壓力。
